Sony WH-1000XM5 Product Page

Web, 3D,

Project Summary

The Sony WH-1000XM5 wireless headphones needed a product landing page to feature the product and direct customers to the Sony online marketplace. The product page aims to informs users about the product in a fun and interactive experience using animation and 3D features for an immersive experince.

Live site

Design

The product site showcases the product design and features of the Sony wireless XM3 headphones. The design reflects elements used in Sony's commerce site and highlights the core features and specs of the product. Interactive elements allow users to change the color and view of the product for a more connected user experience

Models

The model used in this project was downloaded from Sketchfab and designed by artist Zalaneke
